Descripcion del proyecto

Smart Campus Huawei

Nuestro proyecto se trata de crear una plataforma web responsiva con la integración de un Bot de Azure Bot Servicies, el cual ayudara a los alumnos y personal administrativo de la USMP -FIA en tramites e información guiada para presentar en las áreas de Registros Académicos y Tesorería, los tramites a completar por parte del Bot o guías brindadas se almacenaran en una base de datos (dentro de Huawei Cloud), donde también existirá una VPC para comunicación con el App Servicies de Azure, que soportara al Bot y se utilizara la conectividad con las funciones para respuesta concreta o envió de solicitud/captura de información. Dando de esta manera a los usuarios conectados mediante una IoT (Tablet con cámara web).

La eficiencia requerida en este tipo de tramites, ya que actualmente son demorados por intermedio de correos electrónicos y no se tiene un soporte para ser completados. Pensando en el entorno Presencial, se implementara un modulo donde los propios estudiantes mediante reconocimiento facial podrán acceder a consultas y tramites de manera autónoma y rápida, acelerando el proceso de manera intuitiva gracias al Diseño UX que se implementara Lista de servicios • Azure bot service • funciones • Huawei Cloud Herramientas • Git • Azure Devops • Visual Code • PostgreSQL
