Arquitectura del proyecto

Nuestro proyecto se trata de crear un asistente inteligente con Watson para ayudar a resolver consultas a los estudiantes de Universidad San Martin Porres sobre los temas como son los docentes, aulas, cursos y horarios. Explicaremos más a profundidad como es la arquitectura de nuestro proyecto iniciamos con el desarrollador, el cual implementara la programación del sitio web por medio de visual studio code, donde se estructurará el código para su diseño y funcionalidad, iniciado el proceso de creación del sitio, contaremos con el creador de repositorios git para el uso grupal y creación de ramas según se desarrolle el proyecto. Pará deployar el sitio web se necesita el Azure Devops, donde se trabajó conjuntamente las historias de usuario y el proceso grupal de cada rama, para que así se haga el requerimiento de los servicios web con la interfaz del sitio y con su implementación en Internet. Hasta ahí la creación y estructuración del sitio web, ahora el desarrollo descriptivo del servicio Watson. El usuario va a utilizar nuestro asistente Watson, por medio del servicio chatbot, lo hará de manera oral de tal manera que el chatbot lo transcribirá, para lograr la transcripción se contará con una API llamada speech to text, esta API procesará el audio a bits y lo enviará al servicio Watson Assistant donde se hará la transformación de audio a texto, contando con la ia del mismo servicio para la creación de la consulta por parte del usuario, después de la transformación lo plasmara en el servicio de chatbot para q después cumpla con la función del guardado en la base de datos, de esa manera el chatbot se alimente de las consultas de los usuarios, después de la información guardada, pasa y da registro de la respuesta a la web service donde contará con la interfaz previamente creada y con el uso de Internet podrá enviar la respuesta al chatbot con otra funcionalidad del servicio Watson, después enviándola al servicio de Watson Assistant para que la procese, obtenga el contexto y verifique si es la respuesta correcta implementada anteriormente en la base de datos, en la misma nube con el servicio de Watson Assistant, y para finalizar el envío de la respuesta al usuario que realizó la consulta.
