GRUPO SCA ANCHOR

El presente proyecto tiene como objetivo guiar a los usuarios (estudiantes y profesores) a sus respectivos ambientes (salones, auditorio, laboratorios, etc) mediante el uso de realidad aumentada en sus dispositivos móviles con el uso de Azure Anchors. Se utilizará ARWaykit como SDK para poder realizar el mapeo de la facultad y la señalización. Es así, que, al elegir el nombre del curso en un botón plegable, el usuario podrá visualizar unas flechas en dirección al lugar de la FIA USMP que desea ingresar. Del mismo modo, este proyecto contará con un sitio web informativo donde los usuarios podrán acceder a las diferentes páginas proporcionadas por los desarrolladores de este proyecto. La web estará situada en la plataforma de Azure App Services. Ahí podrán encontrar tutoriales de uso, herramientas que fueron utilizadas en el proyecto. Asimismo, la web contará con un chatbot implementado gracias a la herramienta de Azure Bot y QnA Maker para interacciones y absolución de dudas, además hacemos uso de Notifiaction Hubs, una herramienta de Iot de Azure que permitiría al usuario poder recibir notificaciones sobre cualquier actualización que sea liberada, tutoriales sobre uso, guías en pdf, etc.
