Herramientas de ayuda para la gestión de proyectos

Zotero
Herramienta para gestión de citas bibliográficas.
Es libre, abierto y gratuito, multiplataforma, disponible para Windows, Linux y Mac OS.

Azure DevOps
Herramienta para control de versiones.
Cubre el ciclo de vida de un proyecto, ayudando en la gestión, informes, compilación y pruebas necesarias.

GitHub
Herramienta para el alojamiento y control de versiones.
Sistema online de repositorios gratuito. Se utiliza en la creación de código fuente de programas.

VSCode
Herramienta para la creación y edición de código fuente.
Desarrollado por Microsoft para Windows, Linux y Mac OS
Compatible con varios lenguajes de programación.
